## Utilizing Bunkoer

### Secure Your Data Files

Anonymize files effortlessly, starting with CSV formats. Here's how:

```python
from bunkoer.security import SecureFile 

file_path = "/path/to/your/file.csv"
secured_file = SecureFile(file_path)
```
